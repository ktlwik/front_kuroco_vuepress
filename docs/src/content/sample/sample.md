## Kurocoのインストール方法
1. アカウント登録  
```
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
2. 記事グループの作成  
    1. 管理画面にアクセス  
    2. 記事の追加をクリック
